# Guía Interactiva — Canal Directo para Hoteles

**guia.host-feast.online**

Guía interactiva y gratuita para hoteleros independientes en México. Cinco verdades prácticas con calculadoras que funcionan de verdad.

## Contenido

- Calculadora interactiva: ¿Cuánto le regalas a las OTAs?
- Las 5 verdades del canal directo rentable
- Calculadora de BAR mínimo
- Herramientas gratuitas para diagnóstico
- CTA al Kit Hotelero Express

## Stack

- HTML + CSS + JS vanilla
- Sin dependencias externas (solo Google Fonts)
- Deploy automático en Vercel

## Deploy

```bash
vercel --prod
```
